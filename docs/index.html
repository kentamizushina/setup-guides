1. 事前に必要なもの

  • Windows 10（バージョン1809以降）または Windows 11 のパソコン
  • インターネット接続
  • 支払い用のクレジットカードまたはデビットカード
  • Googleアカウント（Gmail）
Windowsのバージョン確認方法: キーボードの Windows + I を押す →「システム」→「バージョン情報」

2. Claude アカウント作成

1
Anthropic アカウントを作成
所要時間: 約3分
2-1

ブラウザ（Chrome推奨）で以下のページを開きます:

https://claude.ai

2-2

「Sign up」（サインアップ）をクリック

Googleアカウントで登録するのが最も簡単です。「Continue with Google」をクリックして、お使いのGmailアカウントを選択してください。

2-3

登録が完了すると、チャット画面が表示されます。これで無料アカウントの作成は完了です。

2
Pro プランにアップグレード
所要時間: 約2分
2-4

画面左下の自分のアカウント名をクリック →「Settings」→「Billing」→「Upgrade to Pro」

2-5

クレジットカード情報を入力します。

料金: 月額 $20（約3,000円）

Claude Code を使うには Pro プラン以上が必要です。無料プランや Team Standard プランでは使えません。

3. Git for Windows インストール

Claude Code が内部で使うツールです。これがないと正しく動きません。

3
Git for Windows をダウンロード・インストール
所要時間: 約5分
3-1

ブラウザで以下のページを開きます:

https://gitforwindows.org

3-2

「Download」ボタンをクリックして、インストーラーをダウンロードします。

3-3

ダウンロードしたファイル（Git-X.XX.X-64-bit.exe）をダブルクリックして実行します。

3-4

インストーラーが起動します。すべてデフォルト（初期設定）のまま「Next」を押し続けてください。設定を変える必要はありません。

最後に「Install」→「Finish」で完了です。

途中で色々な設定画面が出ますが、全て初期設定のままで大丈夫です。変更しないでください。
4
インストールの確認
所要時間: 30秒
3-5

キーボードの Windows キーを押して「PowerShell」と入力し、「Windows PowerShell」をクリックして開きます。

3-6

以下を入力して Enter を押します:

git --version

以下のように表示されれば成功です（数字は多少異なっていてOK）:

git version 2.47.1.windows.1
「'git' は、内部コマンドまたは...」と表示された場合、パソコンを再起動してからもう一度試してください。

4. Claude Code インストール

5
Claude Code をインストール
所要時間: 約3分
4-1

PowerShell を開きます（前のステップで開いたままならそのまま使えます）。

参考: Claude Code 公式ダウンロードページ

4-2

以下のコマンドをコピーして、PowerShell に貼り付けて Enter を押します:

irm https://claude.ai/install.ps1 | iex
コピー&ペーストの方法: 上のコマンドをマウスで選択 → Ctrl + C でコピー → PowerShell画面上で右クリック（自動で貼り付けられます）→ Enter
4-3

インストールが自動で進みます。完了したら、PowerShell を一度閉じて、新しく開き直してください。

4-4

新しいPowerShellで以下を入力して確認します:

claude --version

バージョン番号が表示されれば成功です。

6
Claude Code にログイン
所要時間: 1分
4-5

PowerShell で以下を入力します:

claude
4-6

初回はログインを求められます。ブラウザが自動で開くので、先ほど作成した Claude アカウントでログインしてください。

「Authorize」（許可する）をクリックすると、PowerShell に戻ります。

4-7

ログイン後、チュートリアル画面が出る場合があります。Enter で進めて、最後に /exit と入力して一度終了してください。

5. ワークスペースの準備

7
ワークスペースをダウンロード
所要時間: 約3分
5-1

PowerShell で以下のコマンドを1行ずつ入力して Enter を押します:

# デスクトップに移動 cd ~/Desktop # ワークスペースをダウンロード git clone https://github.com/kentamizushina/ai-workspace-template.git my_workspace
my_workspace の部分は好きな名前に変えてOKです。日本語は避けてください。
5-2

ダウンロードしたフォルダに移動します:

cd my_workspace
5-3

デスクトップに「my_workspace」というフォルダが作成されていれば成功です。

6. 初回セットアップ（/setup）

ここからは Claude Code が対話形式でガイドしてくれます。画面の指示に従うだけで完了します。

8
Claude Code を起動して /setup を実行
所要時間: 15-25分
6-1

PowerShell で、ワークスペースの中にいることを確認してから:

claude

Claude Code が起動し、入力待ちの画面になります。

6-2

以下を入力して Enter を押します:

/setup
6-3 名前と役職

Claude が2つだけ質問します:

  • 「お名前を教えてください」→ 名前を入力
  • 「役職を教えてください」→ 例:「事務」「設計」
6-4 API接続

Claude が1つずつ、外部サービスとの接続を案内します。画面に表示される手順に沿って進めてください。

接続するサービス:

No.サービス用途
1Google (Gmail/Calendar/Drive/Sheets)メール・予定・ファイル管理
2GeminiAI文章生成・PDF読解
3LINE Messaging API顧客連絡
4OpenAIAI補助 + 画像生成
5mnml.ai手書きスケッチ → 建築パース
6不動産情報ライブラリ物件・地域データ
各サービスの設定方法は Claude が画面に詳しく表示します。ブラウザで設定ページを開いて、APIキーをコピーして貼り付けるだけです。
全てのサービスを一度に接続する必要はありません。後から接続を追加できます。まずは Google だけでも使い始められます。

7. 動作確認

9
正しく動くか確認する
所要時間: 約3分
7-1

/setup が完了すると、Claude が自動でテストを行い、以下のような結果が表示されます:

SETUP COMPLETE ============== OS: Windows User: [あなたの名前] ([役職]) APIs: X/9 connected Hooks: OK Rules: OK Ready to use. Type your first request.
7-2

試しに何か聞いてみましょう:

今日のGmailの未読メールを3件教えて

Google接続が完了していれば、メールの内容が表示されます。

セットアップ完了です。これで Claude Code が使えるようになりました。

ショートカットを作成する

毎回 PowerShell を開いてコマンドを打つ必要がないように、デスクトップにショートカットを作ります。

10
デスクトップショートカットの作成
所要時間: 1分（初回のみ）
8-1

PowerShellで、ワークスペースにいる状態で以下を入力します:

claude

Claude Code が起動したら、以下を入力してください:

デスクトップにClaude Codeのショートカットを作って

Claude が自動でショートカットを作成します。

8-2

デスクトップに「Claude Code」というショートカットが表示されます。

次回からは、このショートカットをダブルクリックするだけで起動できます。

毎日の使い方

Claude Code の起動方法（毎回）

  1. デスクトップの「Claude Code」ショートカットをダブルクリック
  2. 黒い画面が開いて、入力待ちになる
  3. あとは日本語で指示するだけ

終了方法

/exit

または Ctrl + C を2回押す

できることの例

やりたいこと入力例
メール確認「今日届いたメールをまとめて」
予定確認「来週の予定を教えて」
書類作成「見積書のテンプレートを作って」
データ整理「このExcelファイルを整理して」
建築計算「この間取りの面積を計算して」

8. 困ったときは

よくあるトラブルと対処法

症状対処法
「'claude' は認識されていません」 PowerShellを閉じて開き直す。それでもダメならパソコンを再起動
「'git' は認識されていません」 Git for Windows を再インストール後、パソコンを再起動
ログイン画面が出ない ブラウザで claude.ai にログインしてから、もう一度 claude を実行
API接続でエラーが出る Claude に「エラーが出ました」と伝えれば、自動で診断・修復を試みます
動きが遅い / 固まった Ctrl + C で中断 → もう一度 claude で起動
何を聞けばいいかわからない 「何ができるか教えて」と入力すればOK

それでも解決しない場合

以下の情報を添えて、担当者に連絡してください:

  • エラーメッセージ（画面のスクリーンショット）
  • どのステップで止まったか（この手順書のステップ番号）
  • パソコンの機種名（わかれば）
