共有API（水信が登録 → 全員で共有）

1. Google Cloud（Gemini API） 共有

Gemini AI（文章生成・PDF読解）のAPIキー。GmailなどのOAuthは各自登録。

aistudio.google.com/apikey

手順: Google AI Studio → APIキー作成 → コピー

GEMINI_API_KEY=xxxxxxxxxx
2. LINE Messaging API 共有

LINE公式アカウントのBot。1チャネルを全員で共有。

developers.line.biz/console

手順: プロバイダー → チャネル → Messaging API → チャネルアクセストークン発行

LINE_CHANNEL_ACCESS_TOKEN=xxxxxxxxxx LINE_CHANNEL_SECRET=xxxxxxxxxx
3. OpenAI API 共有

AI補助（チャット）+ GPT Image 1（汎用画像生成）。1つのAPIキーで全員使える。

platform.openai.com/api-keys

手順: API keys → Create new secret key → コピー

OPENAI_API_KEY=sk-xxxxxxxxxx
5. 不動産情報ライブラリ API 共有

国土交通省提供の物件・地域データ。無料。

reinfolib.mlit.go.jp

手順: 利用者登録 → APIキー発行

REINFOLIB_API_KEY=xxxxxxxxxx
6. Anthropic API 共有

Claude APIの直接呼び出し。自動化スキル・バッチ処理に使用。

console.anthropic.com/settings/keys

手順: Anthropic Console → API Keys → Create Key

ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxx
7. Hugging Face 共有

Whisper（文字起こし）等のAIモデル。無料枠あり。

huggingface.co/settings/tokens

手順: Settings → Access Tokens → New token（read権限）

HUGGINGFACE_TOKEN=hf_xxxxxxxxxx
8. Ideogram API 共有

AI画像生成（テキスト・ロゴに強い）。Plus以上のサブスクが必要。

ideogram.ai/manage-api ideogram.ai/pricing（プラン選択）

手順: Plus($15/月)以上に加入 → API Settings → APIキー生成

IDEOGRAM_API_KEY=xxxxxxxxxx
9. Kling AI API 共有

AI動画生成。テキストや画像から短尺動画を生成。

kling.ai/dev kling.ai/pricing（プラン選択）

手順: Developer Portal → APIキー生成。Standard($6.99/月)以上推奨。

KLING_API_KEY=xxxxxxxxxx

サブスクリプション（水信が契約）

6. Canva ビジネス サブスク

デザインツール。チームメンバーをメールで招待可能。

canva.com/pricing

手順: 既存アカウント → チーム設定 → メンバー招待（メールアドレスで追加）

現在のプラン: Canvaビジネス ¥18,800/年。追加メンバーは1人あたり約¥18,800/年。
7. Adobe Creative Cloud Pro 水信のみ

Photoshop / Illustrator / Premiere Pro / After Effects 他20種以上。水信専用。

adobe.com/jp/creativecloud/plans.html

個人版: 年間一括 ¥102,960/年。年更新。

各自登録（ユーザーごとに必要）

8. Claude Pro 各自

Claude Code を使うために必須。各自のアカウントで契約。

claude.ai

$20/月/人。セットアップガイドのStep 2で案内済み。

9. Google OAuth 各自

各自のGmail / Calendar / Drive にアクセスするため、個別のOAuth認証が必要。

console.cloud.google.com

/setup 実行時に自動で案内されます。

配布手順

APIキー配布の流れ

  • 上記1-5のAPIキーを全て取得
  • .env ファイルにまとめる（下記テンプレート参照）
  • Google Drive の限定共有フォルダに .env を設置
  • 相手にダウンロードリンクを送信
  • 相手が my_workspace/ に .env を配置
  • ダウンロード確認後、Drive の共有を解除

.env テンプレート

# ===== 共有API（水信が登録済み）===== GEMINI_API_KEY=ここに貼る LINE_CHANNEL_ACCESS_TOKEN=ここに貼る LINE_CHANNEL_SECRET=ここに貼る OPENAI_API_KEY=sk-ここに貼る REINFOLIB_API_KEY=ここに貼る ANTHROPIC_API_KEY=sk-ant-ここに貼る HUGGINGFACE_TOKEN=hf_ここに貼る IDEOGRAM_API_KEY=ここに貼る KLING_API_KEY=ここに貼る # ===== 各自設定（/setup で自動案内）===== GOOGLE_CLIENT_ID= GOOGLE_CLIENT_SECRET= GOOGLE_REFRESH_TOKEN=
